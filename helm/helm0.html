Helm - package manager for Kubernetes

Helm packages up all the complex syntax.

Install Helm client on Ubuntu - instructions on Internet
$ helm version
Error: could not find tiller

At this point, you have Helm client installed, but not Helm server..aka, Tiller
$ helm init
Is the command that will install Tiller on Kubernetes - see screen shot
Confirm with:
$ kubectl get pods -n kube-system | grep tiller
and
$ helm version
Will now show both a client and a server version
